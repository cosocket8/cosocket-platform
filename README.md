# COSOCKET - Industrial Machine Rental Platform

## Overview
COSOCKET is a platform connecting machine owners with manufacturers who need industrial equipment on rent.

## Features
- Browse 100+ industrial machines
- Book machines by day/week/month
- WhatsApp integration for instant confirmations
- Provider dashboard to manage listings
- Real-time availability tracking

## Tech Stack
- HTML5, CSS3, JavaScript
- LocalStorage for data persistence
- WhatsApp API for notifications

## Setup Instructions

### Local Development
1. Clone the repository
2. Open `index.html` in your browser
3. No build steps required!

### Deployment to GitHub Pages
1. Push code to GitHub repository
2. Go to Settings → Pages
3. Select branch `main` and `/root` folder
4. Site will be live at `https://yourusername.github.io/repo-name`

### Custom Domain (GoDaddy)
1. Add CNAME file with your domain
2. Add DNS records in GoDaddy:
   - 4 A records pointing to GitHub Pages IPs
   - CNAME record for www
